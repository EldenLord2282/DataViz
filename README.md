# 📊 Data Visualization Dashboard (Python)

A clean and beginner-friendly **Data Visualization Dashboard** built using Python. This project transforms raw data into meaningful visual insights using graphs such as line charts, bar charts, and pie charts.

---

## 🚀 Overview

This project demonstrates how data can be analyzed and visualized effectively using simple Python tools. It is designed to be lightweight, easy to understand, and visually appealing, making it ideal for academic mini-projects and beginners exploring data visualization.

---

## ✨ Features

* 📈 Line Graph for trend analysis
* 📊 Bar Chart for category comparison
* 🥧 Pie Chart for percentage distribution
* 📂 Import data from **Excel (.xls / .xlsx) files**
* 📤 Export processed data to **Excel files**
* 🧹 Clean and minimal output design
* ⚡ Fast and lightweight execution

---

## 🛠️ Tech Stack

* **Python 3**
* **Matplotlib** (for visualization)
* **Pandas** (for data handling and Excel operations)

## ▶️ How to Run

1. Install required libraries:

   ```
   pip install matplotlib pandas openpyxl
   ```
2. Run the script:

   ```
   python dataviz.py
   ```

      or

3. Check the latest release and directly dowload the dataviz.exe file to run directly on your system
